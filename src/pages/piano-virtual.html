ACORDES BÁSICOS: ACORDES MAIORES

Introdução

Bem-vindo ao guia básico de acordes, um recurso para explorar e compreender os acordes de três sons, também conhecidos como tríades. Neste espaço, oferecemos informações sobre a formação dos acordes aumentados.

Piano Virtual

Tutorial: Como Usar o Piano Virtual

1. Piano Virtual

Nessa página, contém o piano virtual. Você deve ver as teclas do piano e alguns botões, como "Iniciar Formação de Acorde", "Piano Virtual", "Tocar Acorde" e "Limpar".

2. Toque nas Teclas do Piano

Cada tecla branca ou preta no piano representa uma nota musical. Clique nas teclas para reproduzir as notas correspondentes. Experimente tocar diferentes teclas para ouvir os diferentes sons.

3. Formação de Acorde (Opcional)

Se desejar formar um acorde, clique no botão "Iniciar Formação de Acorde". Isso permite que você clique em várias teclas do piano para criar um acorde composto por várias notas. As notas serão armazenadas para reprodução posterior.

4. Tocar Acorde

Quando estiver satisfeito com a formação do acorde, clique no botão "Tocar Acorde". Isso reproduzirá todas as notas do acorde simultaneamente, criando um som mais complexo.

5. Limpar Notas (Opcional)

Se quiser começar de novo ou limpar as notas armazenadas, clique no botão "Limpar". Isso reinicia a formação de acorde e apaga todas as notas armazenadas.

6. Explore e Experimente

Sinta-se à vontade para explorar diferentes combinações de notas, acordes e experimentar com os botões disponíveis. O objetivo é se familiarizar com o funcionamento do piano virtual e se divertir explorando diferentes sons musicais.
